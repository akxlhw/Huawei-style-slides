# Framework Selection Guide

Map user intent / scenario to a McKinsey framework.

| Scenario | Primary Framework | Secondary | PPTX Layouts | HTML Slide Types |
|----------|-------------------|-----------|--------------|------------------|
| Recruitment review (quarterly) | Seven-step problem solving | Issue tree + MECE | executive_summary, table_insight, funnel, timeline, action_items | hero, data_table, funnel, timeline, action_grid |
| Channel ROI analysis | MECE + 3C | Pyramid | big_number, data_table, matrix_2x2, table_insight | hero, data_table, matrix_2x2, insight |
| Allocation plan | SCQA + 9-box | MECE | executive_summary, matrix_2x2, data_table, timeline | hero, matrix_2x2, data_table, timeline |
| Executive briefing | Pyramid | MECE | executive_summary, big_number, table_insight | hero, stat_cards, data_table |
| Talent review | 9-box | Talent-to-value | matrix_2x2, data_table, action_items | matrix_2x2, data_table, action_grid |
